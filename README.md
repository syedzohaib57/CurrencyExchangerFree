# CurrencyExchangeFree 💱

CurrencyExchangeFree is a modern Android currency converter app built with Kotlin and Jetpack Compose. The app provides real-time currency exchange rates using the Frankfurter API and allows users to quickly convert between multiple international currencies with a clean and responsive UI.

## ✨ Features

* Real-time currency exchange rates
* Convert between 10+ currencies
* Beautiful Jetpack Compose UI
* Ktor API integration
* MVVM architecture
* Modern Material 3 design
* Dropdown currency selector
* Fast and lightweight
* Built fully in Kotlin

## 🛠 Tech Stack

* Kotlin
* Jetpack Compose
* Ktor Client
* Kotlin Serialization
* MVVM Architecture
* StateFlow
* Coroutines
* Material 3

## 🌍 Supported Currencies

* USD
* PKR
* EUR
* GBP
* INR
* JPY
* AED
* SAR
* CAD
* AUD

## 📡 API Used

* Frankfurter Exchange Rate API
